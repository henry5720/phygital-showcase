# AR V3 Interactive Demo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a project-friendly 1:1 clone of the MindAR official `advanced.html` demo under `/ar/v3`, using official `softmind` assets and maintaining the React project structure.

**Architecture:** Use `ArV3Page` as the React entry point, `ArV3Scene` for the A-Frame markup, and `ar-v3.ts` for the imperative interaction and animation logic.

**Tech Stack:** React 19, A-Frame 1.4.2, MindAR 1.2.2, Vite.
